## Project

極客商城（GeekMall）— 電子產品購物網站

靜態電子產品購物網站，使用 Astro 框架構建，部署於 GitHub Pages。售賣 ASUS、GIGABYTE、Apple、MSI 等品牌約 50 件電子產品。IT114116/ITP4827 網絡分析課程專題，核心目標是建立可被 GA4 追蹤分析的完整購物網站。
<!-- GSD:project-end -->

<!-- GSD:stack-start source:STACK.md -->
## Technology Stack

- **Framework:** Astro（靜態網站生成器）
- **Styling:** CSS（全域樣式 + 元件樣式）
- **Data:** 靜態 JSON 商品資料
- **Analytics:** Google Analytics 4 (GA4)
- **Hosting:** GitHub Pages
- **Build Output:** 純靜態 HTML/CSS/JS
